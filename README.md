Elephly
=======

Тестовое задание. Сделать муху из слона.


Собрать всё
```
make
```

Собрать только исполняемый файл
```
make build
```

Собрать только тесты
```
make build_test
```

Запустить тесты
```
make test
```

Запустить программу с примерами (word.txt, dict.txt)
```
make run
```

Очистить всё
```
make clean
```
