Elephly
=======

Test task.

Example: convert *abc* to *xyz* having dictionary with *abz*, *bbz*, *abx*, *xbz*.

Only one letter can be changed per step.

Solution:

```
abc->abz->xbz->xyz
```


Build all
```
make
```

Build executable
```
make build
```

Build tests
```
make build_test
```

Run tests
```
make test
```

Run program with sample data (word.txt, dict.txt)
```
make run
```

Clean all
```
make clean
```
